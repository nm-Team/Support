# nmBot 2026 年 4 月功能更新

## 2026 年 4 月 26 日 17:02

nmBot 版本号：26.4.0-wine+7332.2000

- 修复了特定情况下消息模板按钮自定义表情留空时消息发送失败的问题。

## 2026 年 4 月 26 日 16:10

### nmBot MCP（Beta 版）和 nmBot Agent Skills

- nmBot 现在支持模型上下文协议。用户可以将 nmBot 连接到自己的代理中，通过代理管理机器人配置。我们还发布了 nmBot Agent Skills 来帮助代理了解 nmBot 功能和配置方式，可在 [GitHub](https://github.com/nm-Team/nmbot-skills) 上访问。

### 消息模板和自定义表情

- 用户现在可以在消息模板中为按钮设置颜色和自定义表情。
- nmBot 现在将在适用的情况下，首选使用拷贝消息的方式回复带有自定义表情的欢乐复读消息。
- 在“在消息模板中使用自定义表情”功能的返回文本中添加了表情 ID 的返回。

### 其他更新和修复

- nmBot 正在测试一项实验性功能，允许短时缓存用户身份来降低 Bot API 负载。这应该能缓解机器人因 API 请求超限导致响应延迟的问题。
- 过塑功能现支持 QuotLy 样式的模板。
- 重构了群友互动功能。
- 更新了 nmBot Copilot 支持的 nmBot 面板页面列表。
- 修复了在受支持情况下，首次打开 nmBot 面板时 iOS 26 和 iPadOS 26 页面效果不正常展示的问题。
- 修复了启用基于 nmBot 智能的新版同频气氛组时，无法正确通过 nmBot 面板关闭“同频气氛组”功能开关的问题。
- 修复了特定情况下，群组管理员可以绕过群主设定的“关键词回复”设置权限来保存消息模板的问题。
- 修复了特定情况下，群组管理员可以绕过群主设定的“入群验证”设置权限来通过或拒绝用户入群的问题。
- 修复了通过特定方式登录 nmBot 面板时超时鉴权不正确工作的问题。
- 修复了发送导出的配置文件失败时，配置文件未正确清理的问题。
- 修复了在 nmBot 面板中打开 nmBot Copilot 对话时，自动聚焦判断条件错误的问题。
