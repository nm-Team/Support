# 如何启动 nmBot 面板

nmBot 面板提供了多种方式启动。

## 在迷你 App 中打开 (推荐)

通过如下方式可在 Telegram 迷你 App 中打开 nmBot 面板：

- Telegram 应用中，对话列表中 nmBot 的“打开 (Open)”按钮。
- Telegram 应用中，和 nmBot 的对话下方的“面板”按钮。
- Telegram 应用中，和 nmBot 的对话中发送 /panel 指令后，返回消息中的链接预览。
- Telegram 应用中，nmBot 资料卡中的“打开 App”按钮。

## 在浏览器中打开

在浏览器中访问 https://nmbot.nmnm.fun 即可打开 nmBot 面板。

## 通过 /panel 指令打开

您可以向 nmBot 发送 /panel 指令来获取面板链接。

您可以根据返回消息中的说明，在[浏览器](#在浏览器中打开)或[迷你 App](#在迷你-App-中打开-推荐)中打开面板。
