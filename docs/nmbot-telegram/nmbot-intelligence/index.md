# nmBot 智能

![nmBot 智能](https://newsroom.nmteam.xyz/usr/uploads/2025/03/1351332983.png)

nmBot 智能是内置于 nmBot 和 nmBot 面板的一系列智能功能。利用机器学习技术，nmBot 智能可以帮助用户更好地管理和使用 nmBot。

## nmBot 智能功能

### nmBot Copilot

[nmBot Copilot](./nmbot-intelligence-chat.md)（原 nmBot 智能聊天）是内置于 nmBot 和 nmBot 面板的个人助理，使管理 nmBot 群组和个人配置变得史无前例地简单。

您只需要用自然语言描述目的，nmBot Copilot 便可以为您配置好相应的功能。

### nmartChat+

nmartChat+ 是 nmartChat 的增强版本，在保留一贯聊天风格的基础上，带来了对对话上下文的增强理解。nmartChat+ 目前为 Beta 版本，面向 nmBot+ 订阅用户提供。

### 骚扰拦截 2.0

nmBot 智能现可驱动[骚扰拦截功能](../group/anti-spam.md)，将骚扰拦截效果提升约 5 倍。群组管理员可以设置 nmBot 智能骚扰拦截的灵敏度，还可启用“智能操作”功能，通过消息的骚扰程度自动决定是仅删除消息还是封禁用户。

### 同频气氛组

基于 nmBot 智能的新版[同频气氛组](../group/reply-channel.md) (Beta 版) 不仅提供更拟人的回复，还能以更自然的方式参与到与其他人的互动中。

### 关键词回复情绪识别

得益于 nmBot 智能，[关键词回复](../group/key-word-reply.md)功能现在支持识别消息文本的情绪，包括仇视言论、侮辱性言论、脏话、暴力言论等。

### 欢乐复读保护

nmBot 智能可帮助阻止“欢乐复读”功能发送意外消息，提高功能的安全性。
