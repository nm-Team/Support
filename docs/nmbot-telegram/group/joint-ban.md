# 联防联控机制

联防联控机制是 nmBot 提供的跨群组用户封禁解决方案，允许多个群组协同封禁同一批用户。

## 功能概述

通过联防联控机制，群组管理员可以：

- 创建联防联控方案，在多个群组中封禁同一批用户
- 通过共享码加入其他群组共享的联防联控方案
- 添加高达 3 个可添加用户到当前群组方案的群组

## 创建联防联控方案

群组管理员可以在 nmBot 面板中创建联防联控方案。创建方案后，您可以将要封禁的用户添加到方案中，所有加入该方案的群组将自动封禁这些用户。

## 加入联防联控方案

其他群组可以通过共享码加入您创建的联防联控方案。加入后，该群组将自动应用方案中的封禁列表。

### 获取共享码

方案创建者可以在 nmBot 面板中生成共享码，并将其分享给希望加入方案的群组管理员。

### 使用共享码加入

群组管理员可以在 nmBot 面板中输入共享码，加入对应的联防联控方案。

## 添加可添加用户的群组

每个群组可以添加高达 3 个可添加用户到当前群组方案的群组。这意味着被添加的群组管理员也可以向方案中添加要封禁的用户。

## <nmbot-plus-icon></nmbot-plus-icon> 自动添加被封禁用户

订阅 nmBot+ 的群组可以在群组中封禁用户时自动将被封禁用户添加到联防联控方案中。

启用此功能后，当群组管理员在群组中封禁用户时，该用户将自动被添加到群组的联防联控方案中，其他加入该方案的群组也将自动封禁该用户。

## 使用场景

联防联控机制适用于以下场景：

- **关联群组管理**：多个相关主题的群组可以共享封禁列表，统一管理骚扰用户
- **社区联盟**：多个独立群组可以组建联盟，协同应对骚扰攻击
- **跨群组骚扰处理**：当骚扰用户在多个群组中活动时，可以一次性封禁所有相关群组
