# nmTeam 账号帮助

欢迎来到 nmTeam 账号帮助中心！在这里，您可以找到有关 nmTeam 账号的各种帮助文档。

## 什么是 nmTeam 账号？

nmTeam 账号是 nmTeam 服务的统一身份认证系统。通过 nmTeam 账号，您可以访问 nmTeam 旗下的各种服务，包括 nmBot Telegram 机器人等。

## 常见问题

??? question "如何重置我的密码？"
    您可以在 [nmTeam 账号中心](https://accounts.nmteam.xyz/) 的登录页面点击"忘记密码"链接来重置您的密码。系统会向您的注册邮箱发送密码重置链接。

??? question "如何更改我的用户名？"
    登录 nmTeam 账号中心后，您可以在"个人资料"页面中修改您的用户名。请注意，用户名更改可能会有一定的限制。

??? question "如何删除我的账号？"
    如果您希望删除 nmTeam 账号，请联系 nmTeam 支持。请注意，账号删除后将无法恢复，且您将失去对所有关联服务的访问权限。
